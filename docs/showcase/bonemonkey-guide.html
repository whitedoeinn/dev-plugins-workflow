Capabilities

Power Features

These aren't just features—they're force multipliers for your development workflow.

🎯

Parallel Research Agents

Four specialized agents run simultaneously: repo-research-analyst scans your codebase, framework-docs-researcher fetches external docs, best-practices-researcher finds patterns, and git-history-analyzer traces evolution.

4 agents parallel execution
🛡

Quality Gates Per Task Type

Each task type maps to specific reviewers. Database changes get data-integrity-guardian. Security code faces security-sentinel. TypeScript files trigger kieran-typescript-reviewer.

6 review agents type-aware
📈

Dependency Resolution

Features declare what they block and what blocks them. Enhanced-ralph builds a dependency graph and executes in topological order—even if the milestone lists them wrong.

Kahn's algorithm cycle detection
🏆

Milestone Mode

Execute an entire milestone with one command. Features run in order, statuses update automatically, and the milestone marks complete when done. Resume from failure with --continue.

--milestone batch execution
⚡

Execution Modes

Fine-tune the workflow with flags: --strict fails on any quality issue (perfect for auth code), --fast skips optional reviews but keeps security gates, --skip-gates runs raw for simple features, and --verbose shows everything.

--strict --fast --skip-gates --continue --verbose --force
Usage

Elegant Invocation

Simple commands that orchestrate complex workflows. Copy, paste, execute.

terminal
1 # Execute a single feature with all quality gates
2 /wdi-workflows:enhanced-ralph auth-system
3
4 # Strict mode for critical features (fails on any issue)
5 /wdi-workflows:enhanced-ralph payment-processing --strict
6
7 # Execute all features in a milestone
8 /wdi-workflows:enhanced-ralph --milestone MILE-002-config-context
9
10 # Resume after a failure
11 /wdi-workflows:enhanced-ralph --milestone MILE-002 --continue
12
13 # Create a new milestone
14 /wdi-workflows:milestone my-milestone
Hidden Gems

The Power You Might Miss

These capabilities aren't obvious at first glance—but they're what separates a tool from a teammate.

✨ OVERLOOKED

Cross-Milestone Dependency Detection

When Feature B in Milestone 2 depends on Feature A in Milestone 1, enhanced-ralph detects this even if the milestones don't declare a dependency. It warns you before you waste time building on incomplete foundations.

✨ OVERLOOKED

Resume from Failure

Milestone execution failed on feature 3 of 5? Run with --continue and enhanced-ralph picks up exactly where it left off. No re-running completed features. No lost progress. Just forward motion.

✨ OVERLOOKED

Implicit Dependency Inference

Even without explicit "Blocked by" declarations, enhanced-ralph analyzes feature descriptions and code patterns to suggest potential dependencies. It asks before assuming—but it notices what humans might miss.
