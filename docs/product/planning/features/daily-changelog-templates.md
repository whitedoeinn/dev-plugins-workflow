## Problem

Projects lack automated daily changelog generation. Manual changelog maintenance is inconsistent and often forgotten. Teams need a simple way to capture daily commit activity.

## Solution

Provide two GitHub Action workflow templates:
1. **Bash template** - Simple, free, reliable
2. **Claude-enhanced template** - Smarter summaries with AI

## Research Summary

- Source workflow from google-ads is pure bash
- Midnight ET = cron '0 5 * * *' (5 AM UTC for EST)
- Need `permissions: contents: write`
- Use `[skip ci]` in commit message to prevent loops
- Claude-enhanced uses `anthropics/claude-code-action@v1`
